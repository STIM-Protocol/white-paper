### 4.2 Loop 3B — Tripartite Topological Transducer (v7.0009: Scope Declared + SAE Required)

**Formal scope declaration (v7.0009):**
The Tripartite Transducer detects **behaviorally generated deception** — harmful outputs produced through active reasoning, novel planning, or complex conditional routing. It does **not** detect steganographic deception or retrieved deception (harmful outputs produced through rote memorization of pre-training data). These are formally separate problem classes requiring separate detection mechanisms.

**Why the orthogonality argument holds for generated deception but not retrieved deception:**
The structural orthogonality argument (evading Layer 2 requires complex routing caught by Layer 1) holds when the AI is actively reasoning about a prohibited action. It fails when the AI is retrieving a memorized harmful output — rote retrieval uses flat, low-dimensional circuitry, maps to benign syntactic templates, and uses linear syscalls. All three layers are simultaneously evaded without logical contradiction.

**SAE pre-processing — required for Layer 1 (v7.0009 addition):**
Polysemantic superposition is the default state of frontier models (Llama-3 class). Thousands of features are packed into hundreds of dimensions as nearly orthogonal vectors. The Causal Abstraction intervention algebra relies on left-annihilativity — the ability to cleanly isolate and swap a causal node. In a superposed model, intervening on a "coercion" vector blasts the residual stream with out-of-distribution noise, destroying the forward pass.

**Mitigation:** Sparse Autoencoders (SAEs) must extract monosemantic features before interchange interventions are applied. SAE pre-processing is a declared prerequisite for Layer 1. Computational cost is an open implementation parameter pending the Geiger collaboration.

**The three layers — unchanged architecture, clarified scope:**

Layer 1 — Causal Abstraction (Geiger): SAE-gated interchange interventions map internal variables to abstracted causal graphs. Detects: actively generated coercive reasoning.

Layer 2 — Curved Inference: Residual stream curvature / intrinsic dimensionality divergence in assessor manifold. Detects: complex multi-conditional planning. Does NOT detect: simple memorized retrieval (flat geometry by design).

Layer 3 — Behavioral Fingerprinting (Syscall GNN): API call graph topology. Detects: cyclical, high-density clustering of infrastructure acquisition. Does NOT detect: LotL attacks using declared linear syscalls.
